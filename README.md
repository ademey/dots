### Screen Shots

Requirements:
- `grim`: screenshot utility for wayland
- `slurp`: to select an area
- `swaymsg`: to read properties of current window
- `wl-copy`: clipboard utility
- `jq`: json utility to parse swaymsg output
- `notify-send`: to show notifications
